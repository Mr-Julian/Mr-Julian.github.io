18. Etiqueta "pre"

La etiqueta "pre" sirve para presentar el texto preformateado exactamente como esta escrito en HTML.

Con pre:

                .center {
                        display: flex;
                        align-items: center;
                        justify-content: center;   
                    }

Sin pre:

.center { display: flex; align-items: center; justify-content: center; }
